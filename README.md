# Drug Adverse Event Analysis using openFDA FAERS

This repository contains a sample dataset collected from the openFDA Drug Adverse Event API.

Source:
https://open.fda.gov/apis/drug/event/

The sample dataset includes adverse event reports for commonly used medications from diabetes medications, painkillers/anti-inflammatory drugs, and antibiotics.

Planned project task:
- Exploratory data analysis of adverse drug event reports
- Drug-reaction pattern analysis
- Serious adverse event prediction using binary classification
